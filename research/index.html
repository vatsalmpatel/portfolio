Linear Desicion Tree Regressor⌗

As Decision Tree Regressor fails to deal with linear relationships, Linear Decision Tree Regressor is build to overcome this limitation by combining the linear regressor with the decision tree regressor. It can do so by using a decision tree structure and a modified node split method, which employs linear regression to better splits the nodes to improve the decision tree performance. The paper discusses the working of the proposed model, how it overcomes the limitations of decision tree regressor and linear regressor. Finally, the model is applied to various datasets, and the performance is compared to that of other regression models.
